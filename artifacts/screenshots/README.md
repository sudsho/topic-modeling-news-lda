# screenshots

Static PNGs exported from the Streamlit app, kept in git so the README
renders without needing to train a model first.

- `topic_words.png` - bar chart of top words for one topic.
- `coherence.png` - c_v coherence vs `k`.

Regenerate by running `make app`, navigating to the relevant section, and
saving the chart via the Streamlit menu (or browser screenshot tool).
